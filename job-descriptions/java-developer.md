# Java Developer

## About the Role

We are seeking a skilled Java Developer to join our engineering team. You will design, develop, and maintain high-quality Java applications that power our core business systems. This role offers the opportunity to work on challenging problems at scale while collaborating with talented engineers.

## Responsibilities

- Design, develop, and maintain robust Java applications using modern frameworks and best practices
- Write clean, testable, and well-documented code following SOLID principles
- Collaborate with cross-functional teams to define and implement new features
- Participate in code reviews and contribute to improving development standards
- Troubleshoot and debug production issues, performing root cause analysis
- Optimize application performance and ensure scalability
- Contribute to architectural decisions and technical documentation
- Mentor junior developers and share knowledge with the team

## Requirements

### Must Have

- 3+ years of professional experience with Java (Java 11+)
- Strong understanding of object-oriented programming and design patterns
- Experience with Spring Framework (Spring Boot, Spring MVC, Spring Data)
- Proficiency in building and consuming RESTful APIs
- Experience with relational databases (PostgreSQL, MySQL) and SQL
- Familiarity with version control systems (Git)
- Understanding of CI/CD pipelines and DevOps practices
- Strong problem-solving skills and attention to detail

### Nice to Have

- Experience with microservices architecture
- Knowledge of messaging systems (Kafka, RabbitMQ)
- Familiarity with containerization (Docker, Kubernetes)
- Experience with cloud platforms (AWS, Azure, GCP)
- Knowledge of reactive programming (WebFlux, Project Reactor)
- Experience with testing frameworks (JUnit, Mockito, Integration Testing)
- Understanding of Agile/Scrum methodologies

## Tech Stack

- Java 17+, Spring Boot 3.x
- PostgreSQL, Redis
- Docker, Kubernetes
- Jenkins, GitLab CI
- Kafka, RabbitMQ
- AWS/Azure

## What We Offer

- Competitive salary and benefits package
- Flexible working arrangements (hybrid/remote options)
- Professional development budget and learning opportunities
- Modern development environment and tooling
- Collaborative and inclusive team culture
- Health insurance and wellness programs

## Interview Process

1. Initial screening call (30 minutes)
2. Technical assessment - take-home coding challenge
3. Technical interview with engineering team (60 minutes)
4. System design discussion (45 minutes)
5. Culture fit conversation with hiring manager (30 minutes)
