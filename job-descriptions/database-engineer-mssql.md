# Database Engineer - Microsoft SQL Server

## About the Role

We are seeking an experienced Database Engineer specializing in Microsoft SQL Server to join our Data Platform team. You will be responsible for designing, implementing, and maintaining our SQL Server infrastructure, ensuring optimal performance, high availability, and data integrity across all database systems.

## Responsibilities

- Design, implement, and maintain SQL Server databases across development, staging, and production environments
- Optimize database performance through query tuning, indexing strategies, and execution plan analysis
- Implement and manage high availability solutions (Always On, Failover Clustering, Log Shipping)
- Develop and maintain backup and disaster recovery strategies
- Create and optimize stored procedures, functions, views, and triggers
- Monitor database health, performance metrics, and capacity planning
- Implement security best practices including encryption, auditing, and access controls
- Collaborate with development teams on database design and data modeling
- Automate routine database administration tasks using PowerShell and T-SQL
- Troubleshoot and resolve complex database issues in production environments
- Document database architecture, procedures, and operational runbooks

## Requirements

### Must Have

- 4+ years of experience as a SQL Server Database Administrator or Engineer
- Expert-level knowledge of T-SQL and query optimization techniques
- Experience with SQL Server 2016+ (preferably 2019 or 2022)
- Strong understanding of database design principles and normalization
- Experience implementing and managing Always On Availability Groups
- Proficiency in performance tuning using execution plans, DMVs, and Query Store
- Experience with backup strategies, DBCC commands, and disaster recovery
- Knowledge of SQL Server security features (TDE, Always Encrypted, Row-Level Security)
- Strong PowerShell scripting skills for automation
- Understanding of Windows Server administration

### Nice to Have

- Microsoft certifications (Azure Database Administrator, SQL Server)
- Experience with SQL Server Integration Services (SSIS) and Reporting Services (SSRS)
- Knowledge of Azure SQL Database and Azure SQL Managed Instance
- Experience with database migration projects
- Familiarity with change data capture (CDC) and replication
- Experience with data warehousing concepts and columnstore indexes
- Knowledge of DevOps practices and CI/CD for database deployments
- Experience with monitoring tools (SolarWinds, Redgate, SentryOne)
- Understanding of GDPR and data compliance requirements

## Tech Stack

- SQL Server 2019/2022
- Always On Availability Groups
- SQL Server Management Studio (SSMS)
- Azure Data Studio
- PowerShell, T-SQL
- SSIS, SSRS, SSAS
- Azure SQL (Database, Managed Instance)
- Redgate SQL Toolbelt
- Windows Server 2019/2022
